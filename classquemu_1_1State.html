Detailed Description

Represents tensor product state of a quantum register with N qubits. For a register with N qubits, the dimension of the joint state is 2 ^ N, where each index represents the complex amplitude corresponding to a particular combination of all N qubits.

quemu::State::State ( const qubit_t  num_qubits)
explicit

Initializes the state with fixed number of qubits. At initialization, the tensor product state will have a complex probability with a value of one in the zeroth index. (Would be measured in the zeroth state with probability one).
